##CrazyFM-Extensions
CrazyFM framework can be easily extended;
Extension is an individual project, that also has its own build configuration.
To use extension in your project based on CrazyFM framework, all you have to do is to add extension compiled SWC or define its source path.
All extensions automatically compile if run CrazyFM global build configuration, which can be found in the **/crazyfm/build/** folder.
